Dynamic Content Filtering

There are many reasons why you might want your users to see only certain parts of the content:

  1. In this sample website, you can test this Filter by clicking on the Filter icon  and select a "Role" from the Group. (Human Resources, Marketing or Software). Notice the TOC changes for those three listings depending on what you select. 

  2. There could be content for entry level or advanced users and you want the user to select what they need.

  3. The content could cover a wide range of products and you want the user to be able to select one or more subsets.

Cross References and Hyperlinks

Hyperlinks from topic to topic or to bookmarks have been part of   from the first versions. Cross References can be thought of as links on steroids.

  • Using hyperlinks, the text of the link never changes. If, for example, you change the name of the target topic, the link remains unchanged.
  • Not so with cross-references, they update automatically. If you create a cross reference to a heading, for example, and then the heading changes, so does the link to it. For example, here is a hyperlink to the Online Banking topic.
